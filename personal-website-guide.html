开始之前

下面从简单入手，制作自己特有风格的网页。你需要准备：一个能制作网页的 AI（如 Kimi）、一个 GitHub 账号、一个 Netlify 账号。

11 步搭建个人网页

01

收集灵感，让 AI 生成网页

自己平时留意到好看的网页，收集起来之后，找到最好的网页，提交给 Kimi 的 Agent 集群，让它模仿生成类似风格的网页；然后将自己已写好的文章上传给 AI 排版，调试生成满意的风格网页。

02

生成 GitHub 可读取的文件夹

网页完成之后，在输入框让 Kimi 生成能让 GitHub 直接访问的文件夹，用于后面上传 GitHub 使用，能直接读取网页的文件格式。

03

创建 GitHub 仓库

下一步就是上传文件夹。如果还没有 GitHub 账号，前往 github.com 注册即可，点击右上角创建 New repository。

04

设置仓库名称

输入 Repository name 后，点击 Create repository 完成设置，进入仓库页面。

05

上传 AI 生成的文件

点击 uploading an existing file，上传 AI 刚生成的 GitHub 文件。

06

拖拽上传并提交

直接拖动文件夹里的文件到空白处上传，不用点击 choose your files 上传；点击 Commit changes 完成项目创建。

07

确认网站文件

如上图，就是能直接读取的网站文件。

08

登录 Netlify

访问 https://www.netlify.com/ 网页托管平台，用 GitHub 账号注册登录。

09

用 GitHub 授权 Netlify

GitHub 授权 Netlify 注册登录。

10

导入 GitHub 仓库

注册完成之后导入刚创建的 GitHub 仓库。

11

自动部署上线

Netlify 自动识别账号内的仓库，点击上传，等待片刻自动部署完成。

🎉 你拥有了自己的数字名片

点击网页即可正常访问，同时可以根据自己需求修改网页的域名。
